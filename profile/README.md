# Краткий гайд по Git и GitHub

## 1. Установка Git
1. Перейдите на сайт [git-scm.com](https://git-scm.com/).
2. Скачайте версию для Windows и установите программу.
3. После установки откройте командную строку (нажмите Win + R, введите `cmd` и нажмите Enter).
4. Введите `git --version` - если увидите версию, установка прошла успешно.

## 2. Клонирование репозитория
Чтобы скачать файлы с GitHub на ваш компьютер:
1. Откройте командную строку.
2. Перейдите в папку, где хотите сохранить файлы: `cd путь\к\папке` (например, `cd Desktop` - рабочий стол).
3. Скопируйте URL репозитория с GitHub.
4. Введите: `git clone https://github.com/username/repository.git`
5. Файлы скачаются в выбранную вами папку.

## 3. Изменение файлов
1. Откройте файлы в любой программе (например, Блокнот или любой другой спец. редактор).
2. Внесите изменения и сохраните файлы.

## 4. Сохранение изменений (Коммит)
1. В командной строке перейдите в папку репозитория: `cd имя_папки_репозитория`.
2. Проверьте статус: `git status` (увидите измененные файлы).
3. Добавьте файлы: `git add .` (точка добавляет все изменения) или `git add имя_файла` для конкретного файла.
4. Создайте коммит: `git commit -m "Ваше сообщение о изменениях"` (например, `git commit -m "Обновил отчет"`).

## 5. Отправка изменений на GitHub (Пуш)
1. Отправьте изменения: `git push origin main` (или в любую другую, в зависимости от ветки, но у нас `main`).
2. Если это первый пуш, введите логин и пароль GitHub (или используйте токен доступа).

## Дополнительные советы
- Для обновления файлов с GitHub: `git pull`.
